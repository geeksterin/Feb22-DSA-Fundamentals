#                           Binary Search Tree
----------------------------------------------------------------------------------------------

***Time Complexity on BST***
----------------------------------------------------------------------------------------------
1. Insertion -> WC: O(N) , BC : O(lgN).
2. Search    -> WC: O(N) , BC : O(lgN).
3. Deletion  -> WC: O(N) , BC : O(lgN).
